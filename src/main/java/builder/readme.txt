建造者模式：实例
    华莱士：有各种食物（Item，接口）、 有各种套餐（Meal，实体类）

    食物(Item)：属性packing是一个接口
        汉堡(Burger)：
            蔬菜堡(VerBurger)：名称，包装(packing)，价格
            鸡肉堡(ChickenBurger)：名称(packing)，包装，价格
        饮料(Drink)：
            Coke：名称，包装(packing)，价格
            Pepsi：名称，包装(packing)，价格

    套餐(Meal)：不同食物的组合
